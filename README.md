# Enchantment Effects

### Because enchantments didn't feel special enough.

Enchantment Effects is a small client-side mod that makes enchantments feel more powerful by adding sounds and particle 
effects. Currently only smite is implemented. Now you can truly smite your enemies!

## Planned Enchantments:
- Smite (Done)
- Sharpness
- Bane of Arthropods
- Fire Aspect
- Knockback
- Thorns
- Possibly Bow Enchantments
